# Install locally

Requirement: Python 3.10+

```shell
python --version
# Python 3.10.8
```

Create a directory for your strategies:
```shell
mkdir strategies
```

Create a Python virtual environment and activate it:
```shell
python -m venv .venv
source .venv/bin/activate
```

Install Cipher:
```shell
pip install cipher-bt[finplot]
```
